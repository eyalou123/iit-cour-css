pseudo-classes

Au passage de la souris :

"a:hover" : Quand la souris est sur le lien" (quand on pointe dessus).

a:hover {
text-decoration: underline; /*Le lien deviendra souligné quand on pointera dessus */
color: green; /* Le lien sera écrit en vert quand on pointera dessus */
}

  • Le pseudo-classe :active permet d’appliquer un style particulier au moment du clic.
  • Le pseudo-classe :visited permet d’appliquer un style au lien déjà visité.

a:active /* Quand le visiteur clique sur le lien */{
background-color: #FFCC66; }

a:hover /* Quand le visiteur pointe sur le lien */ {
text-decoration: underline; color: green; }

a:visited {
color: #00FF00; }

a /* Lien normal */ {
text-decoration: none; color: red; font-style: italic; }
