Style externe :

Pour relier une feuille de styles externe à un fichier HTML, vous utiliserez la balise <link>dans l’entête de ce fichier (ici, la feuille de styles a pour nom design.css) :

HTML: 
<linkrel="stylesheet" href="design.css"/>
  1. La balise <link avertit le navigateur qu’il doit chercher un document situé à l’extérieur de la page HTML.
  2. L’attribut rel= "stylesheet" précise que le document en question est une feuille de style externe
  3. L'attribut rel indique l’emplacement du fichier CSS à utiliser.
HTML: 
<style>
  p {
    margin-left: 20px;
    color: yellow;
  }
  body {
    background-color: #000000;
  }
</style>
⬇️
Here we first create two files: one for CSS (mystyle.css) and another for HTML (homepage.html).
⬇️
HTML: 
<html>
<head>
  <link rel="stylesheet" type="text/css" href="mystyle.css">
</head>
<body>
  <h1>How to add CSS</h1>
  <p>This is my first code</p>
</body>
</html>
⬇️
Add the CSS page to the HTML page using the link tag.

Avantages

  • ✔️ Vous pouvez utiliser le même fichier .css pour plusieurs pages
  • ✔️ Simplifier la maintenance de la mise en forme du site web
